# KI-veileder
Dette prosjektet er kun en mockup-undersøkelse. For implementasjonsdetaljer, se koden i mappen “ai-risk-guide”

## Forutsetning

Du trenger følgende installert i lokalt: 
- Node.js
 - Git

## Kom i gang 
Klone repo og bytt til riktig branch:
```bash
git clone git@github.com:EllaMjaavatn/KI-veileder.git
cd KI-veileder
cd ai-risk-guide
```

## Installer:
```bash
npm install
```

## Start dev-server
```bash
npm run dev
```

## Åpne i nettleseren 
```bash
http://localhost:5173
```
